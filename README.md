# DeepBreath

A digital "secure base" in your pocket for individuals with Autism, ADHD, and sensory processing sensitivities.

## Key Features
- **Stressless Navigation**: AI-powered routes that prioritize quiet and low-crowd paths.
- **Verbal Guidance**: Real-time audio feedback for situational awareness.
- **Panic Support**: One-tap grounding tools, SOS alerts, and "One-Tap Home" navigation.

## Getting Started
1. `flutter pub get`
2. Run on your preferred mobile device.
